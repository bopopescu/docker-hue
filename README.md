Giới thiệu
----------------
Hue là một SQL Workbench mã nguồn mở dùng để tìm kiếm, truy xuất data và visualizing data cho cloud/on-prem Data Warehouses.

Build docker image
----------------
Để build image docker bạn chạy lệnh sau:

```make
   make build
```
